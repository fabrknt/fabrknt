Quick Start

Get up and running in seconds

1

Install

npm install @fabrknt/sdk
2

Import & Configure

import { Fabrknt, Guard } from "@fabrknt/sdk";

const guard = new Guard({
  maxSlippage: 0.1,
  riskTolerance: "moderate",
  mode: "block"
});
3

Execute with Precision

await Fabrknt.execute(transaction, {
  with: guard
});
